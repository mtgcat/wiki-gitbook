## description
- server template based on koa2

## details
- via koa2
- unit test by jest
- use eslint and prettier
- generate document by typedoc 

## scripts
- test: unit test
```bash
npm run test
```

- test: unit test with coverage
```bash
npm run test-cov
```

- build: build
```bash
npm run build
```

- lint,lint:fix : code analysis by eslint and prettier
```bash
npm run lint
npm run lint:fix
```

- docs: genarate document by typedoc
```bash
npm run docs
```

- dev: via nodemon
```bash
npm run dev
```
